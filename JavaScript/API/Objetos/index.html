API Trabajada con Objetos: Documentación

Para el uso de este tipo de APIS,el usuario obtendrá en el sitio web un objeto, siendo éste usualmente parte de una librería (por ejemplo en el caso de GoogleMaps, la propia API nos otorga el objeto "map" y el objeto "marker" que se desplegan) en la pantalla del sitio web después de haber brindado los datos geográficos de interés. En algunos casos se requerirá un pago para acceder a cualquier tipo de servicio (tal es el caso por ejemplo de Twitter), sin embargo algunas otras mientras se cumplan ciertos rubros o ciertas características permanecen de manera gratuita.

Google Maps API:

Para el caso de Google maps, se requerirá tener una cuenta de Gmail así como habilitar los servicios de GoogleCloud, primeramente se mantiene como un servicio gratuito, permitiendo un número específico de vistas al mapa en una cantidad de tiempo. En la documentación de los servicios de GoogleCloud se brinda la información completa de estos servicios y su funcionamiento.
