Documentación
Principales métodos para el uso de Arrays:
Array Original:
Map
Modifica los elementos del arreglo con una condición establecida.
Mostrar los elementos del array divididos a la mitad
Filter
Muestra únicamente los elementos del arreglo que cumplen con una serie de condiciones.
Mostrar los elementos del array mayores a 5
forEach
Itera los elementos del array mostrando el elemento así como su índice
Mostrar a modo de lista cada elemento del arreglo con su índice.
Find
Devuelve la primera coincidencia encontrada del arreglo que cumpla con una condición
Mostrar el primer elemento menor a 5
Sort
Reordena el arreglo dada una condición. Por defecto lo hará de manera ascendente
Reordenar los elementos del arreglo de forma ascendente
Some
Itera el arreglo, si uno de los elementos coincide con una condición, retorna True, en caso contrario retorna False
Mostrar si dentro del arreglo existe un valor menor que 0
Every
Itera el arreglo, si todos los valores coinciden con una condición retorna True como valor, en caso contrario retorna False
Mostrar si todos los valores del arreglo son positivos.
Includes
Determina si dentro del arreglo existe un valor especificado
Mostrar si en el arreglo existe el valor 16
Concat
Unifica dos arreglos sin modificar ninguna de las partes
Unir al arreglo original con un arreglo de valores [a,b,c,d]
Join
Une todos los elementos del arreglo en una cadena de caracteres
Unir el arreglo usando el símbolo "#"
Reduce
Reduce el arreglo haciendo una suma de izquierda a derecha
Mostrar la reducción del arreglo
IndexOf
Busca el índice de un elemento solicitado, de no encontrarlo retorna "-1"
Mostrar el índice del valor 3 en el arreglo
FindIndex
Muestra el índice del elemento que cumpla con una función solicitada
Mostrar el índice del valor que sea mayor que 5 y menor que 8 en el arreglo
Fill
Rellena posiciones específicas de un arreglo con caracteres diferentes
Rellenar de la posición 0 a 2 con el símbolo #
Push
Añade uno o más elementos al final del arreglo
Añadir # al final del arreglo
Pop
Elimina el último elemento del arreglo
Eliminar el último elemento del arreglo
Shift
Elimina el primer elemento del array
Eliminar el primer elemento del arreglo
Unshift
Agrega un elemento al inicio del arreglo
Agregar el símbolo # al inicio del arreglo
Slice
Secciona el arreglo de manera posicional
Mostrar únicamente de la posición 1-2 del arreglo
Reverse
Invierte los valores del arreglo
Mostrar el arreglo a la inversa
lastIndexOf
Busca un elemento del arreglo y regresa el índice del mismo. Trabaja comenzando por el final del arreglo
Mostrar el índice del último elemento que sea mayor que 3
Flat
Unifica el arreglo, incluyendo los sub-arreglos como parte del arreglo principal
Unificar el arreglo: [1,2,3,4,[5,6]]
isArray
Determina si una variable es un arreglo
Mostrar si el arreglo original es un Array
From
Forma un arreglo a partir de una cadena de caracteres
Convertir la cadena "abcdef" en un arreglo
